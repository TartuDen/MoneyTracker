# MVP Core Features

- Google OAuth login (Android)
- Create or join a family via one-time invite code
- Shared lists (groceries, pharmacy, hardware, etc.)
- Real-time list sync with small delay
- Item status + assignment
- Price per item (optional)
- Spending log with categories
- Budget limits with alerts
- Simple analytics (weekly/monthly totals, category breakdown)
- Smart suggestions from frequent items
- Offline mode with auto-sync
